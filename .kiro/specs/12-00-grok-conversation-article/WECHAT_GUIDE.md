# 微信公众号发布指南

## 📱 推荐方案

由于文章较长（约 2万字），建议使用以下两种方式之一：

### 方案一：使用 Markdown 编辑器（推荐）

1. **使用在线 Markdown 转微信工具**
   - 推荐工具：https://md.openwrite.cn/
   - 或者：https://doocs.gitee.io/md/
   
2. **操作步骤**：
   - 打开工具网站
   - 复制 `docs/articles/ai-driven-development-philosophy-and-practice.md` 的全部内容
   - 粘贴到工具的左侧编辑区
   - 右侧会自动生成微信公众号格式
   - 点击"复制"按钮
   - 粘贴到微信公众号编辑器

3. **优点**：
   - 自动转换格式
   - 保留所有样式
   - 一键复制粘贴

### 方案二：手动调整格式

如果你想手动调整，我已经为你准备了格式调整建议：

## 📝 格式调整建议

### 1. 标题层级
- 一级标题（#）→ 使用"标题1"样式，字号 18-20
- 二级标题（##）→ 使用"标题2"样式，字号 16-18  
- 三级标题（###）→ 使用"标题3"样式，字号 14-16
- 四级标题（####）→ 使用加粗，字号 14

### 2. 强调文本
- **粗体文本** → 保持加粗
- *斜体文本* → 改为加粗（微信不支持斜体）
- `代码` → 使用灰色背景或引用样式

### 3. 列表
- 无序列表（-）→ 使用 • 或 ▪ 符号
- 有序列表（1.）→ 保持数字编号

### 4. 分隔线
- `---` → 使用 ━━━━━━━━━━━━━━━━━━━━ 或图片分隔线

### 5. 引用
- `>` 引用块 → 使用引用样式或灰色背景

### 6. 链接
- 微信公众号不支持外链
- 建议在文末添加"阅读原文"链接
- 或者使用二维码

## 🎨 样式建议

### 配色方案
- 主色调：黑色 #000000
- 强调色：深蓝 #1E6FBA 或深红 #C7000B
- 背景色：浅灰 #F5F5F5
- 代码背景：#F0F0F0

### 字体大小
- 正文：14-15px
- 标题1：18-20px
- 标题2：16-18px
- 标题3：14-16px
- 注释/引用：13px

### 行间距
- 正文：1.75-2.0
- 标题：1.5

### 段落间距
- 段前：0.5em
- 段后：0.5em

## 🚀 快速开始

1. 打开 https://md.openwrite.cn/
2. 复制 `docs/articles/ai-driven-development-philosophy-and-practice.md`
