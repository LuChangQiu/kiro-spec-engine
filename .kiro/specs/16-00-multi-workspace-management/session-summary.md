## 关键发现：数据原子性问题

### 问题
当前设计违反 SSOT 原则：
```
~/.kse/workspaces.json  - 工作区列表 + last_accessed
~/.kse/config.json      - active_workspace + preferences
```

### 解决方案
采用单一配置文件：
```
~/.kse/workspace-state.json  - 所有工作区相关数据
```
