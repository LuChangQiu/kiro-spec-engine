# Spec 12-00: Grok 对话文章整理与发布

## 目标

整理与 Grok 的对话内容，发布到 GitHub 和其他技术平台。

## 文件说明

- **draft.md** - 对话原始内容草稿（用户填写）
- **article.md** - 整理后的发布版本（AI 生成）
- **metadata.json** - 文章元数据（标题、标签、发布平台等）

## 工作流程

1. ✅ 创建 Spec 目录和草稿文件
2. ⏳ 用户复制 Grok 对话内容到 `draft.md`
3. ⏳ AI 整理对话，生成 `article.md`
4. ⏳ 创建 GitHub 文档文件（`docs/articles/`）
5. ⏳ 准备 GitHub Discussions 发布内容
6. ⏳ 推荐其他发布平台
