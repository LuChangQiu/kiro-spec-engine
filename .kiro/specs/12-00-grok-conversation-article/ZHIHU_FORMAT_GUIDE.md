# 知乎格式优化指南

> 知乎对 Markdown 的支持有限，需要做格式调整

---

## 知乎不支持的格式

1. ❌ **引用块中的加粗** - `> **文字**`
2. ❌ **复杂的表格**
3. ❌ **HTML 标签**
4. ❌ **某些 emoji 组合**
5. ❌ **嵌套的列表格式**

---

## 知乎支持的格式

1. ✅ **标题** - `#`, `##`, `###`
2. ✅ **加粗** - `**文字**`
3. ✅ **斜体** - `*文字*`
4. ✅ **列表** - `-` 或 `1.`
5. ✅ **代码块** - ` ``` `
6. ✅ **链接** - `[文字](url)`
7. ✅ **简单引用** - `>`
8. ✅ **分隔线** - `---`

---

## 优化建议

### 1. 简化引用块

**原格式**：
```markdown
> **作者**：heguangyong  
> **对话对象**：Grok (xAI)  
> **日期**：2026-01-24
```

**优化为**：
```markdown
作者：heguangyong  
对话对象：Grok (xAI)  
日期：2026-01-24
```

### 2. 简化标题

**原格式**：
```markdown
## 🎯 核心观点
```

**优化为**：
```markdown
## 核心观点
```

### 3. 简化列表

**原格式**：
```markdown
- **范式转变**：AI 从"被动投喂"到"主动探寻"
```

**优化为**：
```markdown
1. 范式转变：AI 从"被动投喂"到"主动探寻"
```

---

## 发布步骤

1. 复制原文内容
2. 在知乎编辑器中粘贴
3. 手动调整不支持的格式：
   - 删除 emoji
   - 简化引用块
   - 调整列表格式
4. 预览检查
5. 发布

---

## 快速修复方案

如果格式混乱，可以：
1. 删除当前文章
2. 使用"富文本编辑器"而非 Markdown 模式
3. 分段复制粘贴，逐段调整格式
