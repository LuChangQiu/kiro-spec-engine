# 发布说明：v1.46.2

发布日期：2026-02-14

## 本版亮点

- 新增 KPI 可观测命令，用于量化交付结果：
  - `kse value metrics snapshot`
  - `kse value metrics baseline`
  - `kse value metrics trend`
- 新增首次使用脚手架命令：
  - `kse value metrics sample`
- 提升操作体验：
  - 失败提示新增“下一步命令”指引，降低上手阻力。
- 强化产品优势表达：
  - README、快速入门、文档索引已统一“结构化 + 编排 + 可量化”定位。
- 统一入口文档：
  - `START_HERE.txt`、`INSTALL_OFFLINE.txt`、`docs/OFFLINE_INSTALL.md` 已同步到当前版本流程。
- 提升 npm 包洁净度：
  - 已排除 Python 字节码产物，减少无效打包内容。

## 为什么这个版本重要

这个版本把 kse 从“流程工具”进一步升级为“可量化交付系统”：

- 可按周生成机器可读 KPI 证据；
- 可比较 baseline 与 trend 做风险判断；
- 可输出 Day30/Day60 门禁可复用结果。

## 3 分钟快速验证

```bash
kse adopt
kse spec bootstrap --name 01-00-demo-feature --non-interactive
kse value metrics sample --out ./kpi-input.json --json
kse value metrics snapshot --input ./kpi-input.json --json
```

## 升级说明

- 运行时要求保持不变：Node.js >= 16。
- 现有工作流保持向后兼容。
- 若你已使用 value metrics，`sample` 为新增可选能力，不影响原流程。
