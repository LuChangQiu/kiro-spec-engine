## 📋 使用说明

### 快速安装（3 步）

```bash
# 1. 将 kiro-spec-engine-1.24.1.tgz 复制到目标机器

# 2. 安装
npm install -g kiro-spec-engine-1.24.1.tgz

# 3. 验证
kse --version
# 应显示: 1.24.1
```

### Windows 用户特别注意

如果遇到 "npm 命令不可用" 错误：

```powershell
# 临时添加 Node.js 到 PATH
$env:Path = "C:\Program Files\nodejs;$env:Path"

# 然后执行安装
npm install -g kiro-spec-engine-1.24.1.tgz
```

**原因**: Windows PowerShell 会话可能未刷新环境变量

**永久解决**: 重启终端或添加 Node.js 到系统 PATH

## 🎯 功能演示

### 嵌套仓库扫描

```bash
# 扫描所有嵌套仓库（默认）
kse repo init

# 查看仓库状态（显示父子关系）
kse repo status --verbose

# 只扫描顶层仓库
kse repo init --no-nested
```

### 知识管理

```bash
# 初始化知识库
kse knowledge init

# 添加知识条目
kse knowledge add

# 搜索知识
kse knowledge search "Git"

# 查看统计
kse knowledge stats
```
